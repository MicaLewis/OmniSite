Particle System

Based on Particle Life by CodeParade on Youtube, but re made in Processing. Each type of particle has a randomized set of relationships to other particles. Some particles are repeled from particles that are attracted to them, creating a chase condition. Each particle is subject to drag so that it doesn't gain speed indefinately. Rather than existing in a box it is run on a topology similar to a torus. Not only can particles cross from edge to edge of the screen, but so can the forces that repel and attract them.
